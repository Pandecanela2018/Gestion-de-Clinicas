# Gestion de Clinicas
Front End, Backend y base de datos 

## **Pasos para extraer los archivos del main a su rama.**

1- Clonar el repositorio en Visual.

2-Cambiarse a su rama perteneciente.

3- Abrir la terminal

4- colocar el siguiente script git merge origin (Usuario corresponiente)

## **Pasos para inicializarlo correctamente.**

1-Abrir la terminal

2-Colocarse en la carpeta /FastAPIex

3- Crear el entorno virtual con el script **python -m virtualenv .venv**

4-Activar el entorno virtual con este script **.\.venv\scripts\activate**

5- Descargar la siguiente dependencia: **pip install "fastapi[standard]"** en la carpeta **\FastAPIex\API**

## **Pasos para iniciarlo el archivo**
1- Colocar en la terminal **fastapi dev main.py**
